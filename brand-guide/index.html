How to Import

Every HTML page must include this single link tag. Adjust the path based on directory depth.

<!-- Root level -->
<link rel="stylesheet" href="assets/css/ekwa-brand.css">

<!-- One level deep (e.g. /verticals/dentistry/) -->
<link rel="stylesheet" href="../../assets/css/ekwa-brand.css">

<!-- Two levels deep (e.g. /verticals/dentistry/tools/) -->
<link rel="stylesheet" href="../../../assets/css/ekwa-brand.css">

<!-- Three levels deep (e.g. /verticals/dentistry/head-of-sales/email-sequences/) -->
<link rel="stylesheet" href="../../../../assets/css/ekwa-brand.css">
